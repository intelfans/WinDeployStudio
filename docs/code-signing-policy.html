代码签名政策

Code signing policy

WinDeploy Studio 的 Windows 发布包遵循公开源码、可追溯构建和人工发布审批流程。

WinDeploy Studio Windows releases follow a public-source, traceable-build, and manual release-approval process.

签名服务

免费代码签名由 SignPath.io 提供，证书由 SignPath Foundation 提供。

签名证书属于 SignPath Foundation。SignPath Foundation 可以根据其项目条款暂停服务或撤销证书；签名不代表 Microsoft 对本项目提供担保或官方认可。

构建与发布

签名文件必须由 WinDeploy Studio 自有公开源代码和构建脚本生成。发布前会核对版本、源代码、构建产物和 SHA-256；每个发布版本都需要人工批准。

本项目不会使用本项目订阅为其他项目的上游二进制文件签名。运行时使用的 Windows 系统组件和未重新分发的系统库不属于本项目签名范围。

团队角色

当前公开仓库由 GitHub 用户 intelfans 维护。角色映射如下：

  • Authors / Committers：intelfans，负责维护源代码和构建脚本。
  • Reviewers：项目维护者负责审阅来自非提交者的变更。
  • Approver：intelfans，负责在签名请求中确认发布内容。

SignPath 团队权限、GitHub 权限和发布审批权限应保持一致，并启用多因素认证。

用户数据与隐私

标准版的可选 AI、诊断和联网搜索功能只在用户主动使用或确认后发起远程请求。安装程序提供禁用远程 AI 和联网搜索的隐私选项。详情请参阅隐私政策。

发布页面

安装包、更新说明和 SHA-256 信息发布在应用下载页面及公开 GitHub Releases 中。

Signing service

Free code signing provided by SignPath.io, certificate by SignPath Foundation.

The signing certificate belongs to SignPath Foundation. SignPath Foundation may pause the service or revoke the certificate under its project terms. A signature does not imply endorsement or warranty by Microsoft.

Builds and releases

Signed files must be produced from WinDeploy Studio's own public source code and build scripts. Before release, the version, source, build output, and SHA-256 are checked; every release requires manual approval.

This project does not use its subscription to sign upstream binaries from other projects. Windows system components and system libraries that are not redistributed are outside this project's signing scope.

Team roles

The public repository is maintained by GitHub user intelfans. The role mapping is:

  • Authors / Committers: intelfans, responsible for source code and build scripts.
  • Reviewers: project maintainers review changes proposed by non-committers.
  • Approver: intelfans, responsible for approving signing requests.

SignPath, GitHub, and release-approval permissions should remain aligned, with multi-factor authentication enabled.

User data and privacy

The standard edition's optional AI, diagnostic, and web-search features make remote requests only after the user invokes or confirms them. The installer provides a privacy option to disable remote AI and web search. See the Privacy Policy for details.

Release pages

Installers, release notes, and SHA-256 information are published on the app download page and public GitHub Releases.

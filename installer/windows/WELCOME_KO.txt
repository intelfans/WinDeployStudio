주요 기능:

  🖼️ 이미지 라이브러리
    엄선된 Windows 이미지와 다운로드 링크 탐색
    
  💿 설치 미디어 생성 도구  
    Windows 설치 미디어를 만들거나 부팅 가능한 Linux ISOHybrid 이미지 쓰기
    
  🚀 To Go 환경
    휴대용 Windows 환경과 구조 검사를 통과한 x64 Ubuntu/casper, Debian Live/Kali,
    Deepin Live 프로필용 실험적 영구 Linux To Go를 만듭니다.
    Linux To Go는 GPT 대상 디스크가 있는 x64 UEFI 시스템에서만 지원되며, UEFI+MBR 및 레거시 BIOS는 지원되지 않습니다.
    실제 부팅과 재부팅 후 영구 저장 검증이 여전히 필요합니다
    
  📊 네이티브 드라이브 테스트
    순차, 4K, 멀티스레드 및 지속 쓰기 성능 측정

═══════════════════════════════════════════════════════════════

시스템 요구 사항:

  • Windows 10/11 (64비트)
  • USB 3.0+ 드라이브 (권장)
  • 설치 및 디스크 작업 시 관리자 승인 필요

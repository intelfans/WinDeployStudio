주요 기능:

  🖼️ 이미지 라이브러리
    엄선된 Windows 이미지와 다운로드 링크 탐색
    
  💿 설치 미디어 생성 도구  
    Windows 설치 미디어를 만들거나 부팅 가능한 Linux ISOHybrid 이미지 쓰기
    
  🚀 Windows To Go 환경
    휴대용 Windows 작업 환경을 만듭니다. Linux 휴대용 환경은 향후 릴리스에서 제공될 예정입니다.
    
  📊 네이티브 드라이브 테스트
    순차, 4K, 멀티스레드 및 지속 쓰기 성능 측정

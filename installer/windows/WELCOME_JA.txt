WinDeploy Studio をお選びいただきありがとうございます！

═══════════════════════════════════════════════════════════════

主な機能：

  🖼️ イメージライブラリ
    厳選された Windows イメージとダウンロードリンクを閲覧
    
  💿 インストールメディア作成ツール  
    Windows メディアを作成、または起動可能な Linux ISOHybrid を書き込み
    
  🚀 To Go 環境
    ポータブル Windows 環境と、構造チェックを通過した x64 Ubuntu/casper、
    Debian Live/Kali、Deepin Live 向けの実験的な永続化 Linux To Go を作成します。
    Linux To Go は GPT ターゲットディスクを使用する x64 UEFI システムでのみサポートされ、UEFI+MBR とレガシー BIOS はサポートされません。
    実機起動と再起動後の永続化の検証が引き続き必要です
    
  📊 ネイティブドライブテスト
    シーケンシャル、4K、マルチスレッド、持続書き込みを測定

═══════════════════════════════════════════════════════════════

システム要件：

  • Windows 10/11（64ビット）
  • USB 3.0+ ドライブ（推奨）
  • インストールとディスク操作には管理者の承認が必要

Основные функции:

  🖼️ Библиотека образов
    Просмотр образов Windows и ссылок для скачивания
    
  💿 Создание установочного носителя  
    Создание носителей Windows или запись загрузочных Linux ISOHybrid-образов
    
  🚀 Windows To Go
    Создание переносной среды Windows. Переносные среды Linux запланированы для будущей версии.
    
  📊 Нативный тест накопителя
    Проверка последовательной, 4K, многопоточной и длительной записи

═══════════════════════════════════════════════════════════════

Системные требования:

  • Windows 10/11 (64-бит)
  • USB 3.0+ накопитель (рекомендуется)
  • Для установки и операций с дисками требуется подтверждение администратора

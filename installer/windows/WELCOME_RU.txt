Основные функции:

  🖼️ Библиотека образов
    Просмотр образов Windows и ссылок для скачивания
    
  💿 Создание установочного носителя Windows  
    Создание установочных носителей Windows из файлов ISO
    
  🚀 Windows To Go
    Создание портативных рабочих пространств Windows на USB-накопителях
    
  🌍 Многоязычная поддержка
    Полный китайский и английский интерфейс

  • Windows 10/11 (64-бит)
  • USB 3.0+ накопитель (рекомендуется)
  • Требуются права администратора

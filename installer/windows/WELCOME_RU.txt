Основные функции:

  🖼️ Библиотека образов
    Просмотр образов Windows и ссылок для скачивания
    
  💿 Создание установочного носителя  
    Создание носителей Windows или запись загрузочных Linux ISOHybrid-образов
    
  🚀 Среды To Go
    Создание переносных сред Windows и экспериментальных постоянных Linux To Go для
    структурно проверенных профилей x64 Ubuntu/casper, Debian Live/Kali и Deepin Live.
    Linux To Go поддерживается только на системах x64 UEFI с целевыми дисками GPT;
    UEFI+MBR и традиционный BIOS не поддерживаются; всё ещё нужны проверка реальной загрузки и сохранения данных после перезагрузки
    
  📊 Нативный тест накопителя
    Проверка последовательной, 4K, многопоточной и длительной записи

═══════════════════════════════════════════════════════════════

Системные требования:

  • Windows 10/11 (64-бит)
  • USB 3.0+ накопитель (рекомендуется)
  • Для установки и операций с дисками требуется подтверждение администратора

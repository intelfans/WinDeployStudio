## 🚀 WinDeploy Studio v1.1.1

### ✨ New Features
- No major new user-facing feature in this release
- This update focuses on Image Center wording, LTSC metadata, and release consistency

### ⚙️ Improvements
- Refined **Windows 10 Enterprise LTSC** descriptions across all 11 supported languages
- Improved LTSC wording so descriptions explain the long-term servicing channel clearly
- Updated the **Windows 10 Enterprise LTSC 123 Cloud mirror** link
- Updated app, Windows metadata, installer, scripts, README, and release notes to **v1.1.1**

### 🛠 Fixes
- Fixed awkward wording where Windows 10 Enterprise LTSC could read as if it was "based on itself"
- Fixed the same LTSC wording issue in English, French, German, Spanish, Portuguese, Russian, Arabic, Korean, Japanese, Simplified Chinese, and Traditional Chinese
- Fixed stale v1.1.0 installer output naming and documentation references

### 📌 Notes
This is a small polish release for trust, clarity, and consistency.

The Image Center behavior remains unchanged: Official Microsoft entries open Microsoft websites, Community and LTSC entries use the mirror-based download flow.

## 🚀 WinDeploy Studio v1.1.1

### ✨ 新增功能
- 本版本没有新增大型用户功能
